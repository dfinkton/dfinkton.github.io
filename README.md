# Darryl Finkton Jr. - Author Website

A modern, readable website showcasing the writings of Darryl Finkton Jr., optimized for both human readers and AI accessibility.

## Content

- **3 Books** split into chapters/parts
- **8 Essays** on poverty, UBI, and social change
- **7 Peer-reviewed articles** on medical research
- **Videos** including informational content and mini-documentaries

## Tech Stack

- **Hugo** - Static site generator
- **GitHub Pages** - Free hosting with automatic deployment
- **Custom theme** - Clean, minimal design with semantic HTML

## Local Development

```bash
# Install Hugo (if not already installed)
winget install Hugo.Hugo.Extended

# Run local server
hugo server

# Build for production
hugo
```
